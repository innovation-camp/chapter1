## How to run

### Python version
```shell
$ python -V
  Python 3.8.9
```

### clone repository
```shell
$ git clone https://github.com/innovation-camp/heonchaegjulge.git
```

### into the repository
```shell
$ cd heonchaegjulge
```

### install packages
```shell
$ pip install -r requirements.txt
```

### start app
```shell
$ flask run
```
